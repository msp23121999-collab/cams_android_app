# 08. Module Documentation

### Auth Module
- **Responsibilities:** User authentication, JWT generation, RBAC enforcement.
- **Dependencies:** Passlib, python-jose.

### Academic Module
- **Responsibilities:** Curriculum, Courses, Subjects, and Enrollments.

### Attendance Module
- **Responsibilities:** Logging student presence, geofencing, and smart-classroom integration.

### Financial Module
- **Responsibilities:** Budgeting, Payroll, Deductions, and PFs.

### Legal Module
- **Responsibilities:** Moot court session handling, Activity Points.
