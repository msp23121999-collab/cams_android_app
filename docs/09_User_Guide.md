# 09. User Guide

## Step-by-Step Instructions
### 1. How to Login
- Launch the CAMS Enterprise app.
- Enter your registered email address and password.
- Click **Login**.
- If your account possesses multiple roles, you will be prompted with a **Role Selection** screen.

### 2. Navigation
- Use the **Bottom Navigation Bar** to switch between core portal features (Dashboard, Profile, Notifications).
- Use the **Side Navigation Drawer** for deeper modules (Attendance, Exams, Reports).

### 3. Portal Specifics
- **Students:** Click on "My Attendance" to view metrics.
- **Faculty:** Click on "Smart Classroom" to begin a lecture session.
